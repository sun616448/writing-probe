One-time setup

Enable screen reader support in Google Docs

Google Docs hides its text from browser extensions by default. One setting unlocks it — you only need to do this once per browser.

1
Open any Google Doc (or use the button below to open one now).
2
In the menu bar, click Tools → Accessibility settings
3
Check the box next to Turn on screen reader support, then click OK.
4
Reload the Google Doc tab. Thinking Probe will start working immediately.
This setting gives Thinking Probe read-only access to your document text so it can generate probe questions. It does not give the extension any ability to edit your document.
